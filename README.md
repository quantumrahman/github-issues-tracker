## 📝 Questions & Answers

### 1️⃣ What is the difference between `var`, `let`, and `const`?

* **`var`** :
  1. Function scoped.
  2. Can be redeclared and reassigned.
  3. Hoisted to the top of the function.
* **`let`** :
  1. Block scoped.
  2. Cannot be redeclared in the same scope, but can be reassigned.
* **`const`** :
  1. Block scoped.
  2. Cannot be redeclared or reassigned.
  3. Immutable reference (object or array values can change, but the reference cannot).

---

### 2️⃣ What is the spread operator `(...)`?

The spread operator expands an array or object into individual elements or properties. Useful for copying or merging arrays and objects.

---

### 3️⃣ What is the difference between `map()`, `filter()`, and `forEach()`?

* **`map()`** : Transforms each element of an array and returns a new array.
* **`filter()`** : Returns a new array containing elements that satisfy a condition.
* **`forEach()`** : Iterates over an array but does not return a new array.

---

### 4️⃣ What is an arrow function?

A shorter syntax for writing functions. Does not require the `function` keyword. Automatically binds `this` in certain contexts.

---

### 5️⃣ What are template literals?

Template literals allow embedding variables and expressions inside strings. Use backticks ``` instead of quotes.
